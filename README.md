# MedPal – AI Health Diary

## Overview
MedPal is a prototype health diary that converts user symptoms into structured summaries using AI.

## Features
- Symptom input
- AI-based summary (prototype)
- Clean UI

## Tech Stack
- HTML, CSS, JS
- Google Gemini API
- Firebase
